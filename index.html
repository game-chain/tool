GameChain
简介

GameChain游戏公链是一个基于区块链底层的开放公链项目，所有的游戏参与者及开发者都可以共同参与开发及完善公链，并且所开发游戏的内测、公测、正式发行都能在GameChain上完美执行，GameChain目的是打通游戏通证化改造，使传统的游戏开发者与游戏运营商运能够方便地将游戏资产通证化、快速实现通证的公开交易，以及遵循通证经济理念进行游戏运营。同时让游戏用户能公开透明的了解游戏规则，在相对公平的环境下游戏。GameChain是游戏开发者、游戏运营商、游戏玩家、价值投资人、专业做市商和其它参与者所共同构建的一个公平、和谐、有趣的游戏产业生态。

发行计划

总量 1,000,000,000 GAME

创世节点 每个节点10,000,000 GAME 锁仓

市场流通 10,000,000 GAME

挖矿 900,000,000 GAME

关于挖矿：

第一年挖矿产出8%*1000,000,000即8,000,000GAME；

第二年挖矿产出6%* (1+8%)*1000,000,000即4,800,000GAME；

第三年挖矿产出4%，按以上通胀公式，即4,480,000GAME;

第四年挖矿产出2%，按以上通胀公式，即2,345,600GAME;

第五年以后每年挖矿产出1%,直到10亿枚GAME发行完毕为止
